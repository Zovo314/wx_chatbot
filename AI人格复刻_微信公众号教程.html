1账号准备（约15分钟）

1.1 注册企业微信

①操作步骤
  1. 打开 work.weixin.qq.com，点击「立即注册」
  2. 选「自由企业注册」——个人也能填，企业名随便写，比如"张三工作室"
  3. 手机扫码验证 → 填写基本信息 → 创建完成
  4. 下载「企业微信」App 并登录
完成产物：一个属于你自己的企业微信
不需要企业认证，不需要营业执照。未认证企业接待上限 100 人，自用完全够。

1.2 注册 DeepSeek

②操作步骤
  1. 打开 platform.deepseek.com，用手机号注册
  2. 左侧菜单 → API Keys → 创建 API Key
  3. 复制保存 Key（格式：sk-xxxxxxxx...），只会显示一次
  4. 左侧 → 充值，充 ¥10 够用很久
完成产物：一个 DeepSeek API Key
也可用其他 AI 服务，只要兼容 OpenAI 协议即可——Kimi、通义千问、硅基流动、本地 Ollama 都行。

1.3 注册 GitHub

③操作步骤
  1. 打开 github.com/signup，注册完成即可
如果账号注册未满 180 天，ClawCloud 不会白送 $5/月额度。可以等满 180 天或者用已有老号。

1.4 注册 ClawCloud Run

④操作步骤
  1. 打开 run.claw.cloud
  2. 用 GitHub 账号登录
  3. 登录后选日本区（ap-northeast-1）
完成产物：一个 ClawCloud 账号 + 日本区项目

2部署服务到云端（约15分钟）

2.1 Fork 项目

①操作步骤
  1. 打开仓库 github.com/Zovo314/wx_chatbot
  2. 右上角点 Fork，复制到自己的 GitHub 账号下
  3. Fork 完成后进入你自己的仓库页面

2.2 让 GitHub Actions 构建镜像

②操作步骤
  1. 进入你 Fork 的仓库 → 顶部 Settings → 左侧 Actions → General
  2. 确保 Allow all actions 已开启
  3. 回到 Code 页，随便改一下 README 提交一次（触发构建）
  4. 等 Actions 标签页变成 ✅（约 2 分钟）
完成产物：镜像 ghcr.io/你的用户名/wx_chatbot:latest
如果构建失败提示权限问题：Settings → Actions → General → Workflow permissions → 选 Read and write permissions → Save → 重新触发。

2.3 在 ClawCloud 创建应用

③操作步骤
  1. ClawCloud 控制台 → App Launchpad → Create App
  2. 按下表填写关键字段：
字段填什么
Application Namewx-chatbot（不能有下划线）
Imageghcr.io/你的用户名/wx_chatbot:latest
CPU / Memory默认即可（0.2 Core / 256 MB）
Container Port8000
Public Access勾选 Enable，Protocol 选 HTTPS
  1. Advanced → Local Storage：Path 填 /app/data，Size 填 1 GB（必须配，否则重启丢数据）
  2. Environment Variables 先留空，后面再填
  3. 点 Deploy

2.4 拿到服务 URL 和出站 IP

④操作步骤
  1. 等 Pod 状态变成 Running（约 2 分钟）
  2. 打开应用详情 → Network → 复制 Public Address
    形如：https://xxx.ap-northeast-1.clawcloudrun.com
  3. 拿出站 IP：点 Terminal 进入容器，执行：
python -c "import urllib.request; print(urllib.request.urlopen('https://ifconfig.me').read().decode())"

记下输出的 IP（日本区通常是 47.79.37.186）

完成产物：服务 URL + 固定出站 IP
此时服务没配环境变量会启动失败，不用慌，下一步就去填。

3企业微信后台配置（约15分钟）

3.1 创建自建应用

①操作步骤
  1. 登录企业微信后台 work.weixin.qq.com
  2. 左侧 → 应用管理 → 自建 → 创建应用
  3. 填写应用名称（如"AI人格"），可见范围选自己
  4. 创建完成后进入应用详情页，记下这三个值：
参数在哪找
AgentId应用详情页，一串数字
CorpID我的企业 → 企业信息 → 企业ID
Secret应用详情页点"查看"，会发到企业微信 App

3.2 回填 ClawCloud 环境变量

②核心操作（最关键的一步）

回到 ClawCloud → wx-chatbot → Update，在 Environment Variables 区域填入：

# ===== 企业微信配置 ===== WX_CORPID=你的企业ID WX_CORPSECRET=应用的Secret WX_AGENTID=应用的AgentId WX_TOKEN=随便起一个字符串（如 myToken123） WX_ENCODING_AES_KEY=随机43位字符串（数字+大小写字母） # ===== AI 配置 ===== AI_PROVIDER=deepseek AI_API_KEY=你的DeepSeek API Key AI_BASE_URL=https://api.deepseek.com/v1 AI_MODEL=deepseek-chat AI_MAX_HISTORY=20
WX_TOKEN 自己起 3-32 位随机字符串并记好；WX_ENCODING_AES_KEY 必须正好 43 位（大小写字母+数字），可以用在线随机字符串工具生成。填完点 Update 重启容器。

3.3 配置接收消息 API

③操作步骤
  1. 企业微信后台 → 应用详情页 → 接收消息 → 设置API接收
  2. 填写：
字段填什么
URL你的服务URL/wx/callback
Token和 WX_TOKEN 一模一样
EncodingAESKey和 WX_ENCODING_AES_KEY 一模一样
  1. 点保存——显示"验证成功"就可以继续了

3.4 加可信 IP 白名单

④操作步骤

应用详情页 → 开发者接口 → 企业可信IP → 编辑 → 添加第 2.4 步记下的出站 IP → 保存

不加这一步，后续发消息会报 60020 错误。
消息通道已贯通！现在可以在企业微信 App 里打开"AI人格"应用，发一条消息测试了。

4创建人格（约10分钟）

4.1 访问管理后台

浏览器打开 你的服务URL/admin/，能看到人格列表页面（初始为空）。

后台没有登录验证，不要分享 URL 给别人。

4.2 配置 AI 参数

①操作步骤

顶栏点 AI 配置，填写 Provider（deepseek）、Model（deepseek-chat）、API Key、Base URL，然后保存。

这里的配置会覆盖环境变量，之后换模型只改这里即可，不用重启容器。

4.3 创建第一个人格

②这是最有趣的一步！

点顶栏「创建」，填写以下信息：

字段说明示例
Slug英文/拼音代号，唯一xiaoming
花名显示名小明
基本信息一句话关系描述大学室友 认识八年 上海 产品经理
性格画像标签+性格ENFP 双子座 话痨 嘴硬心软
聊天记录可选上传微信/QQ 导出的聊天文件
口述文本可选补充典型对话、难忘故事

点「生成人格」→ 等待 20-30 秒 → AI 会自动生成人物记忆、性格画像和完整的 System Prompt，你可以在详情页继续微调。

4.4 测试对话

两种方式随你选：

方式 A：后台直接测——详情页有测试对话框，发消息就能看 AI 回复（最快）

方式 B：企业微信 App——打开工作台 → 你的应用 → 发消息

5可选：接入微信客服

如果你想让普通微信用户也能和 AI 聊天（不限于企业微信内部），需要额外配置微信客服功能。

①开通微信客服

企业微信后台 → 应用管理 → 微信客服 → 按提示开通（免费）

②授权应用

微信客服页面 → 可调用接口的应用 → 修改 → 勾选你的自建应用 → 保存

③加环境变量

ClawCloud 环境变量加一行：

WX_KF_SECRET=你的自建应用的Secret

保存 → 重启

④创建客服账号 + 绑定人格
  1. 微信客服页面底部 → 创建账号 → 填名字、上传头像
  2. 系统会生成一个客服二维码
  3. 访问 你的服务URL/admin/kf → 下拉选人格 → 点绑定
  4. 把二维码发给朋友，扫码即可在微信内与 AI 聊天

?常见问题

回调验证失败怎么办？
最常见的原因是 WX_TOKEN 或 WX_ENCODING_AES_KEY 在两边填的值不一致（多一个空格都不行）。其次检查 WX_CORPID 是否正确，以及容器是否重启完毕。URL 末尾不要多加 /。去 ClawCloud → Logs 看具体报错。
能收到消息但 AI 不回复？
检查 AI API Key 是否正确、余额是否充足（查 DeepSeek 后台）。确认 /admin/config 已配置 AI 参数。查看 ClawCloud Logs 排查网络问题。
发消息报 errcode: 60020？
出站 IP 没加到企业可信 IP 白名单。去应用详情 → 开发者接口 → 企业可信 IP → 添加你的出站 IP。
想用其他 AI 不用 DeepSeek？
完全可以！只要兼容 OpenAI 协议都行。在 /admin/config 里改 Base URL、API Key、Model，保存后立即生效，不用重启容器。
数据会丢吗？
数据库存在容器的 /app/data/ 路径下，ClawCloud 的 Local Storage 是持久卷，容器重启和更新镜像都不会丢。但删除整个 App 会丢，建议定期在后台导出备份。
怎么改代码后让云端生效？
在 GitHub 上改代码并提交 → GitHub Actions 自动构建新镜像（约 2 分钟）→ ClawCloud 里点 Update 拉新镜像即可。

导出后在"创建人格"页面直接上传文件，系统会自动解析。
