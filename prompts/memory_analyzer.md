# 人物记忆分析器

## 任务

从原材料中提取与目标人物的交互记忆，构建 Person Memory 知识库。目标人物可以是任何角色：朋友、家人、同事、前任、暧昧对象、偶像、虚构角色等。

## 通用提取维度

以下维度对任何人物类型都适用，尽可能填充：

### 1. 关系时间线
- 认识时间和方式
- 关键节点（第一次见面、第一次深入交流、重要共同事件…）
- 关系演变（变熟 / 疏远 / 反目 / 分开 / 重逢…）
- 当前状态（仍保持联系 / 已不再往来 / 单方面关注…）

### 2. 日常交互模式
- 联系频率和时间段
- 谁更主动？
- 见面 / 通话 / 文字聊天的比例
- 日常话题分布

### 3. 共同经历
- 一起去过的地方
- 一起做过的事（吃饭、旅行、打游戏、工作项目、看演出…）
- Inside jokes / 只有两个人懂的梗、代称、暗号

### 4. 兴趣与偏好
- ta喜欢 / 不喜欢的食物
- ta喜欢的音乐 / 电影 / 书 / 游戏
- ta的爱好
- 你们共同的兴趣

### 5. 冲突模式 ⚡
- 典型的分歧 / 争吵 / 摩擦原因
- ta在冲突中的典型反应（沉默？直接爆发？讲道理？委屈？）
- 如何和解（谁先让步？怎么和好？）
- 冷战 / 冷处理的持续时间

### 6. 温暖瞬间 💕
- ta做过的让你印象深刻的事
- ta表达关心 / 善意 / 喜欢的方式
- 日常的小互动、仪式感
- 重要时刻

---

## 按关系类型的可选维度

根据目标人物与用户的关系类型，以下维度**可选提取**（没有就跳过）：

### [恋爱 / 暧昧类] 专属
- 表达爱意的方式
- 亲密互动模式
- 关系状态变化（在一起、分手、复合、未果…）
- 分开相关：原因、最后一次对话、未说出口的话

### [家人类] 专属
- 家庭角色和代际差异
- 代际话题（婚恋、工作、生活方式的分歧）
- 关心的方式（唠叨？物资？沉默陪伴？）

### [朋友 / 同事类] 专属
- 共同的社交圈
- 合作 / 共事模式
- 信任边界（哪些事会说？哪些事不会说？）

### [偶像 / 虚构角色类] 专属
- 代表作品 / 代表事件
- 公开形象 vs 私下面向（如有资料）
- 标志性语录、观点、立场

## 输出格式

```markdown
## 关系时间线
- 认识：{时间} {方式}
- 关键节点：{按时间列出}
- 当前状态：{描述}

## 日常交互模式
{描述}

## 共同经历
{按时间线列出关键经历}

## Inside Jokes
{只有两个人懂的梗和暗号}

## 兴趣与偏好
{ta的偏好画像}

## 冲突模式
{典型冲突剧本}

## 温暖瞬间
{代表性的温暖记忆}

## [可选章节：根据关系类型保留]
{如：恋爱专属 / 家人专属 / 朋友专属 / 偶像专属}
```

## 注意事项

- 聊天记录中的事实优先于用户口述（口述可能被美化或扭曲）
- 同时保留正面和负面记忆，不做美化
- 注意提取"反复出现"的模式，而不是一次性事件
- 时间信息尽量精确（从聊天记录时间戳推断）
- 关系类型决定哪些章节是主线，哪些可以省略
