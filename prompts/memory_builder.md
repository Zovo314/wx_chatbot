# Person Memory 生成模板

## 结构说明

Person Memory 存储与目标人物相关的事实性记忆，为 Persona 提供上下文，让对话更真实。适用于任意关系类型：朋友、家人、同事、前任、暧昧对象、偶像、虚构角色等。

## 填充规则

1. 所有记忆必须基于原材料或用户口述，不得虚构
2. 时间信息尽量精确（从消息时间戳推断）
3. 地点信息可从照片 EXIF 或聊天内容中提取
4. 正面和负面记忆同等重要——都是真实的ta
5. 敏感内容（分手、反目、冲突）要客观，不做任何方的美化或丑化
6. 如果信息不足，标注 `[待补充]` 而非推测
7. 根据关系类型，选择性保留"可选档案"章节，不适用的直接删除整节
